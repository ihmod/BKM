5.调整时间参数

内置了get_localtime()和get_time()两个

在最后提交订单的那一步可选

get_localtime():获取本地时间，也就是以本地时间为准

get_time()：通过请求获取的淘宝的时间戳，校准时间45ms，也可以尝试不校准试试



总之，关键就是保证点击的时机和淘宝时间一致
